# TS-ES-playground-7

## 1. dependency
elasticsearch version 7.x
@elastic/elasticsearch 7.x
  

## 2. 환경 설정
.env.sample 내용 작성 후 .env로 새로 저장

## 3. 실행
```
  yarn install
  yarn run
```
